# SETUP INSTRUCTIONS

### Windows
1. Install [WSL](https://learn.microsoft.com/en-us/windows/wsl/install).
2. Open your new Linux terminal.
3. Run: `sudo apt update && sudo apt install sshpass vim git -y`

### ChromeOS
1. Enable the [Linux Development Environment](
   https://support.google.com/chromebook/answer/9145439?hl=en). 
2. Open the Linux terminal.
3. Run: `sudo apt update && sudo apt install sshpass vim git -y`

### macOS                                         
1. Install [Homebrew](https://brew.sh/). Read the terminal output carefully 
   and execute the "Next steps" to add it to your PATH.
2. Open your terminal.
3. Run: `brew install bash sshpass vim git`
